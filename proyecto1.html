Zazil.click

Descripción:

Zazil.click es una tienda en línea construida para la organización Todas Brillamos, quienes se dedican a combatir la pobreza menstrual. La plataforma desarrollada consta de una página web de escritorio, desde donde es posible administrar las ventas de la organización, pero también consta de una aplicación móvil, desde donde los usuarios pueden consultar el catálogo dinámico, crear sus cuentas, añadir productos al carrito, realizar sus compras y monitorear sus pedidos. Una función adicional, llamada “Nuestra para ti”, también crea un vínculo especial entre la organización y el cliente final, al mostrarle la contribución de sus compras en el combate a la pobreza menstrual.

Tecnologías usadas:

  • React
  • Node.js
  • Stripe
  • Kotlin

Habilidades desarrolladas:

  • Desarrollo full stack
  • Desarrollo de aplicaciones móviles para Android con Kotlin
  • Arquitectura Model View View-Model
  • Seguridad en aplicaciones móviles
  • Operaciones CRUD
