Cropopoly 2 - Ghost Town

Descripción:

Cropopoly 2 - Ghost town es un sistema multi-agentes con gráficas computacionales que simula la interacción de diferentes agentes en una ciudad. Esta simulación es de gran utilidad para comprender las configuraciones viales que pueden ser problemáticas y que pueden generar embotellamientos. Además de crear el sistema multi-agentes en Python, también se modelaron los elementos 3D en Blender para ser presentados en un escenario en Unity, en donde se renderea la simulación generada en Python mediante la librería MESA.

Tecnologías usadas:

  • Python
  • Blender
  • Unity

Habilidades desarrolladas:

  • Programación orientada a objetos con Python
  • Modelado de sistemas multi-agentes con MESA y Python
  • Modelado 3D con Blender
  • Render de simulación en Unity
