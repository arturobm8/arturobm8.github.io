Curso de stack MERN

Descripción:

He buscado perfeccionar mis habilidades de desarrollo full Stack a través del curso “Web Personal MERN Full Stack: MongoDB, Express, React y Node”, en donde aprendí valiosos consejos y estrategias para desarrollar un sitio web transaccional con un back end desarrollado con JavaScript y también con un front end desarrollado en React, así como diversos middlewares y librerías para promover la usabilidad y la seguridad de la plataforma.

Tecnologías usadas:

  • MongoDB
  • Express
  • React
  • Node.js

Habilidades desarrolladas:

  • Implementación de bases NoSQL
  • Desarrollo back end
  • Desarrollo front end
  • Arquitectura full stack
  • Medidas básicas de seguridad web
  • Despliegue de sitio web
